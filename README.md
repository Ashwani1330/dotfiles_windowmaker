### The Stack
* **OS:** Arch Linux (CachyOS)
* **WM:** Window Maker
* **Terminal:** Ghostty
* **Editor:** Neovim
* **File Manager:** Thunar
* **Theme:** Dracula (GTK3) + Violet-Blue (WM)
* **Dockapps:** `wmsystemtray`, `wmnd`, `volumeicon`, `wmgtemp`
